# TT5 child

Her er et child theme til Twenty Twenty-five. De vigtigste filer er:

* functions.php - importerer stylesheet mv. fra TT5.
* style.css - definerer child themet for Kontrolpanelet.

## Parent temaet

Forudsætningen for at et child theme fungerer er at parent theme er til stede.
